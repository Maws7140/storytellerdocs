# Commands

All commands are accessible from the command palette (`Ctrl/Cmd + P`). You can assign hotkeys to any command in **Settings > Hotkeys**.

## Dashboard and Stories

| Command | Description |
|---------|-------------|
| Open dashboard view | Opens the main dashboard panel |
| Create new story | Creates a new story container |
| Refresh story discovery | Rescans the vault and updates story metadata |

## Templates

| Command | Description |
|---------|-------------|
| Open story template gallery | Browses built-in and custom story-world templates |
| Open entity template library | Manages entity-level templates |
| Reload custom templates | Refreshes the template list from disk |
| Save note as template | Saves the current entity note as a reusable template |

## Characters

| Command | Description |
|---------|-------------|
| Create new character | Opens the character creation modal |
| View all characters | Lists all characters in the active story |
| Generate character sheet | Opens the character sheet preview for the current note |

## Locations

| Command | Description |
|---------|-------------|
| Create new location | Opens the location creation modal |
| View all locations | Lists all locations in the active story |

## Events and Timeline

| Command | Description |
|---------|-------------|
| Create new event | Opens the event creation modal |
| View timeline | Opens the timeline in a modal |
| Open timeline panel | Opens the timeline as a persistent workspace panel |
| Manage timeline eras | Opens the era manager |
| Manage timeline tracks | Opens the track manager |
| Detect timeline conflicts | Runs conflict detection and shows results |
| Generate events from scene tags | Creates timeline events from tagged scene content |
| Auto-generate tracks | Creates tracks based on existing characters and locations |
| Create timeline fork | Creates an alternate timeline branch |
| View timeline forks | Lists all timeline forks |
| View timeline conflicts | Shows existing conflict detection results |

## Groups

| Command | Description |
|---------|-------------|
| Create group | Opens the group creation modal |
| View groups | Lists all groups in the active story |
| Rename group | Renames a selected group |
| Delete group | Deletes a selected group |

## Manuscript and Scenes

| Command | Description |
|---------|-------------|
| Create new chapter | Opens the chapter creation modal |
| View all chapters | Lists all chapters in the active story |
| Create new scene | Opens the scene creation modal |
| View all scenes | Lists all scenes in the active story |
| Go to next scene | Navigates to the next scene in draft order |
| Go to previous scene | Navigates to the previous scene in draft order |
| Compile manuscript | Compiles the active draft into a single note |
| Create draft | Creates a new manuscript draft |

## Maps and Gallery

|---------|-------------|

## Story Board and Analytics

| Command | Description |
|---------|-------------|

## Import

| Command | Description |
