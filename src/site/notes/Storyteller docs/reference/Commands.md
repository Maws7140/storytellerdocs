# Commands

All commands are available from the command palette (`Ctrl/Cmd + P`) unless noted otherwise.

## Important note

**Campaign Play** is currently opened from the **swords ribbon icon** in the left sidebar, not from a dedicated command palette command.

## Dashboard and stories

| Command | Description |
|---------|-------------|
| Open dashboard view | Opens the main Storyteller dashboard |
| Create new story | Creates a new story container |
| Refresh story discovery | Rescans the vault for stories and story metadata |

## Templates

| Command | Description |
|---------|-------------|
| Open story template gallery | Browse built-in and custom story templates |
| Open entity template library | Manage entity-level templates |
| Reload custom templates | Refresh the custom template list from disk |
| Save note as template | Turn the current entity note into a reusable template |

## Characters

| Command | Description |
|---------|-------------|
| Create new character | Open the character modal |
| View all characters | List characters in the active story |
| Generate character sheet | Open the character sheet preview/export flow |

## Locations

| Command | Description |
|---------|-------------|
| Create new location | Open the location modal |
| View all locations | List locations in the active story |

## Events and timeline

| Command | Description |
|---------|-------------|
| Create new event | Open the event modal |
| View timeline | Open the timeline in a modal |
| Open timeline panel | Open the persistent timeline panel |
| Manage timeline eras & periods | Edit eras / periods shown behind the timeline |
| Manage timeline tracks | Edit filtered timeline tracks |
| Detect timeline conflicts | Run conflict detection and review the results |
| Generate timeline from tags | Create timeline events from tagged notes |
| Auto-generate timeline tracks | Create starter tracks from existing entities |
| Create timeline fork | Create an alternate timeline branch |
| View timeline forks | Review saved timeline forks |
| View timeline conflicts | Review saved conflict results |

## Groups

| Command | Description |
|---------|-------------|
| Create group | Open the group modal |
| View groups | List groups in the active story |
| Rename group | Rename a selected group |
| Delete group | Delete a selected group |

## Manuscript and scenes

| Command | Description |
|---------|-------------|
| Create new chapter | Open the chapter modal |
| View all chapters | List chapters in the active story |
| Create new scene | Open the scene modal |
| View all scenes | List scenes in the active story |
| Go to next scene | Move to the next scene in the active draft |
| Go to previous scene | Move to the previous scene in the active draft |
| Compile manuscript | Run the active draft through its compile workflow |
| Create draft | Create a new manuscript draft |

## Maps and gallery
